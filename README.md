# Gpt voice graber

Голосовой мост: диктовка ChatGPT → системный ввод куда угодно.

Диктуешь голосом — ChatGPT в фоновом табе Firefox расшифровывает речь — и текст одной клавишей появляется в окне, где ты работаешь (VS Code, редактор, мессенджер). Сам в браузер не переключаешься.

**В чём польза:**
- **Качество ChatGPT, но везде.** Распознавание с пунктуацией и многоязычностью — не только в чате, а в любом приложении.
- **Отдельный платный диктовщик не нужен** — задействуешь диктовку, которая уже есть в ChatGPT (по идее — как WhisperFlow).
- **Не отрываешься от работы.** Триггер — одна клавиша из любого приложения; фокус, ненадолго уйдя в браузер, сам возвращается к тебе.

## Как работает

По хоткею **Ctrl+Win** C#-сервер командует расширению кликнуть диктовку в ChatGPT; расширение читает расшифровку и шлёт её серверу по WebSocket, а сервер вставляет текст в активное окно.

```
┌────────────┐  автовставка  ┌──────────────┐  mic/stop/clear (WS)  ┌─────────────┐
│ активное   │ ◄──────────── │  C#-сервер   │ ────────────────────► │  Firefox    │
│ окно (цель)│               │  VoiceBridge │ ◄──────────────────── │  расширение │
│            │               │  хук+вставка │  text/recording (WS)  │  (ChatGPT)  │
└────────────┘               └──────────────┘                       └─────────────┘
```

## Как пользоваться

1. Запусти сервер и загрузи расширение (см. ниже). На кнопке расширения — зелёная точка ●.
2. Работай в своём приложении. Когда нужно надиктовать:
   - **Ctrl+Win** — старт. Окно Firefox на миг мелькнёт (так Firefox разрешает запись), фокус сам вернётся к тебе. Говори.
   - **Ctrl+Win** ещё раз — стоп. Через 1–3 с распознанный текст вставится в твоё окно.
   - **Ctrl+Win+Y** — то же, но текст ещё и останется в буфере обмена.

## Сборка и запуск

**Сервер** (нужен .NET 10 SDK):

```powershell
cd server-csharp\VoiceBridge
dotnet run
```

Окно консоли — это и есть сервер (там же логи), `Ctrl+C` — выход. Если `dotnet run` ругается отказом доступа к порту — выполни один раз в админ-консоли:
`netsh http add urlacl url=http://localhost:17890/ user=ДОМЕН\Пользователь`.

**Расширение** (Firefox): `about:debugging#/runtime/this-firefox` → **Load Temporary Add-on…** → `extension-firefox/manifest.json`. Зелёная точка ● на кнопке = WS подключён. Подробнее — в [extension-firefox/README.md](extension-firefox/README.md).

## Как это устроено

- **Управление.** Низкоуровневый хук клавиатуры ловит Ctrl+Win (старт/стоп); клавиша «+буфер» — по скан-коду, поэтому не зависит от раскладки.
- **Старт записи.** Firefox не пишет микрофон, пока окно в фоне, — сервер на миг поднимает окно браузера, запись стартует, и фокус сразу возвращается тебе.
- **Вставка.** Текст кладётся в буфер и вставляется синтезом Ctrl+V (`SendInput`) — так он доходит до Electron/Chromium и корректно вставляет любой Unicode.
- **Связь.** Расширение ↔ сервер по WebSocket на `localhost:17890`.
