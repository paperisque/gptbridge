# fonts/ — приватные шрифты индикатора

Положи сюда `.ttf` / `.otf` — индикатор статуса подхватит их **без установки в систему**
(GDI+ `PrivateFontCollection`, см. `StatusOverlay.PickFont`). Файлы из этой папки
копируются в выход при сборке и имеют приоритет над системными шрифтами.

Подпись пилюли ищет первую установленную/подложенную семью из цепочки
`StatusOverlay.TextFontChain` (сейчас: **Google Sans → Roboto → Segoe UI**).
Имя семьи должно совпадать с именем в самом файле шрифта (не с именем файла).

Сами файлы шрифтов в гит НЕ коммитим (`.gitignore`): они могут быть проприетарными
(Google Sans/Product Sans — фирменный шрифт Google, свободно распространять нельзя).
